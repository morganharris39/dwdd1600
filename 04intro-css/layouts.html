Layouts

What are Keyboard Layouts?

A Keyboard layout refers to the both the size of the keyboard and the style. Keyboard sizes include: 65%, 85%, and others. Styles include your standard rectangular keyboards, as well as alice and split keyboards.

Size

Full sized/100%
A 100% keyboard is one that includes all 104 keys, such as arrow keys, funcion keys, and the numpad.
TKL/85%
A TKL (ten-key-less) keyboards has all the same keys as a Full Sized Keyboard, except it is missing the numpad. These have 87 keys.
75%
These are very similar to TKLs. However, the home cluster is aligned vertically and is squished very closely to the arrow keys, making this keyboard size more compact.
65%
These keyboards are super compact since theres no home cluster or function key row.
60%
60% keyboards are the second smallest since the arrow keys are no longer present. These are very popular keyboards too.
40%
These are the smallest keyboards that you can type on, since they only feature letters and your basic functions on the side like shift and enter. There isn't even a number row at the top.
Macro Pads/Number Pads
These are very small, normally programmable keyboards that serve as either a separate numpad or a shortcut pad.

Styles

  • Standard - These are your basic rectangular keyboards
  • Alice - Alice keyboards have split the keys down the middle, and then angled them inwards. This makes the keyboard more ergonomic
  • Split - These are similar to Alice keyboards, except the entire keyboard is split down the middle. Each side is connected by a cable in the middle and can also be positioned to best fit your comfort needs.
  • Ortholinear - Most keyboards have each row of keys offset from each other, almost laid similarly to bricks. Ortholinear keyboards don't have this; each key is stacked on top on top of each other so they align perfectly.
